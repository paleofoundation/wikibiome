# Key Claims

- Computational drug repositioning identified novel candidate compounds for anti-TNF refractory CD using CMap-based analysis
- Cyclosporin, vorinostat (HDAC inhibitor), and piperlongumine showed the highest enrichment scores for CD gene expression signatures
- Cluster 1 genes (158 genes) were enriched in chemokine signaling, cytokine-cytokine receptor interaction, and TLR signaling pathways
- HDAC inhibitors and NF-kB inhibitory agents represent promising drug classes for refractory CD
- Three biologically related pathway groups (chemokine, cytokine-receptor, TLR signaling) are central to anti-TNF refractory CD pathogenesis

# Methods/Findings

- **Study type**: In silico drug repositioning using transcriptome data (GSE100833) from anti-TNF refractory CD patients
- **Methods**: Co-expression analysis (cogena); STRING protein-protein interaction network; KEGG pathway analysis; Connectivity Map (CMap) drug signature matching
- **DEGs identified**: 260 differentially expressed genes in 4 clusters; 193 up-regulated, 67 down-regulated
- **Top drug candidates**: CP-690334-01, cyclosporin, vorinostat, withaferin A, chelidonine, etofenamate, piperlongumine, rimexolone
- **Cluster 2 drugs**: Metronidazole, ethosuximide, famprofazone, 2,6-dimethylpiperidine, betonicine, clomipramine
