# Key Claims

- Serum metabolomics using GC/MS can discriminate pancreatic cancer from healthy volunteers with high sensitivity (86.0%) and specificity (88.1%) in the training set
- A diagnostic model using four metabolites (xylitol, 1,5-anhydro-D-glucitol, histidine, inositol) achieved AUC of 0.9286 in the training set
- The metabolomics-based model showed higher sensitivity (77.8%) in resectable stage pancreatic cancer (0-IIB) compared to conventional markers CA19-9 (55.6%) and CEA (44.4%)
- The model showed lower false-positive rate (17.4%) in chronic pancreatitis compared to CA19-9 (30.4%)

# Methods/Findings

- **Study design**: Multi-institutional study with training set (43 PC, 42 healthy volunteers) and validation set (42 PC, 64 HV+CP)
- **Analytical platform**: GC/MS-based metabolomics targeting water-soluble metabolites; 159 compounds detected, 45 subjected to semiquantitative evaluation
- **Key metabolites**: 18 metabolites significantly altered in PC vs. healthy; 1,5-anhydro-D-glucitol had highest individual AUC (0.83499)
- **Diagnostic performance**: Training AUC 0.9286; validation AUC 0.76004; sensitivity 71.4%, specificity 78.5% in validation
- **Chronic pancreatitis discrimination**: 16 metabolites also altered between CP and HV, but model distinguished PC from CP (AUC 0.87)
