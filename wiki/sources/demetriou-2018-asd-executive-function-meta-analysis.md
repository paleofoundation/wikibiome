# Overview

Large meta-analysis of **235 studies (N = 14,081; ASD = 6,816, Controls = 7,265)** examining executive function (EF) performance in ASD. Confirms moderate overall EF deficit across all subdomains with high true heterogeneity.

# Key Findings

## Overall Executive Function Deficit

- Overall Hedges' g = 0.48 (95% CI: 0.43-0.53, P < 0.001)
- True heterogeneity was large (I^2 = 75.5%)
- Effect size remained significant after excluding questionnaire-based measures (g = 0.49)

## Subdomain Analysis

- **Concept Formation**: g = 0.49 (92 studies)
- **Mental Flexibility**: g = 0.48 (38 studies)
- **Fluency**: g = 0.45 (56 studies)
- **Planning**: g = 0.55 (51 studies, largest effect)
- **Response Inhibition**: g = 0.46 (103 studies)
- **Working Memory**: g = 0.47 (70 studies)
- No significant differences between EF subdomains

## Moderator Analyses

- **Age**: significant moderator only for Working Memory domain
- Year of Publication: small but significant moderating impact for Concept Formation, Fluency, Planning
- Clinical sensitivity of individual EF measures was low (majority did not achieve criterion)

## Clinical Implications

- EF impairment is **pervasive but not fractionated** across development
- Moderate effect sizes suggest EF difficulties are real but not sufficiently specific for diagnostic use
- BRIEF informant measures showed greatest clinical utility
