# Zeng 2025 — Copper, Iron and Trace Elements in Fibromyalgia (Mendelian Randomization)

## Key Findings

- **Copper** is causally associated with **increased FM risk** (OR = 1.095, p = 0.018) in two-sample Mendelian randomization.
- **Iron** shows an **inverse causal association** — lower iron status raises FM risk (OR = 0.440, p = 0.012).
- **No significant causal signal** for calcium, zinc, selenium, magnesium, or folate in multivariate MR.
- Large sample (n > 400,000) strengthens causal inference despite the MR framework's limitations.

## Methods

Two-sample Mendelian randomization using GWAS summary statistics. 2,689 FM cases + 397,508 controls. Multivariate MR to adjust for co-varying micronutrient exposures.
