# Gut Microbiome Depletion Impairs Gui-Shen-Wan's Ovarian Function Restoration

## Key Claims

- Antibiotic-mediated gut microbiome depletion abolished beneficial effects of Gui-Shen-Wan on ovarian function
- Mice with intact microbiome showed better treatment outcomes
- Gut microbiome is pivotal for mediating TCM therapeutic effects on ovarian function
