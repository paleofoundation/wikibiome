# Peritoneal Microbial Features and Tumor Markers as OC Diagnostic Tools

## Key Claims

- OC patients had a unique peritoneal microbial profile; 18 microbial features highly specific to OC pathology
- Inclusion of microbial features with tumor markers improved diagnostic accuracy
