# Yao 2024 — Gut Microbiota and Menstrual Disorders (MR)

## Key Findings

- ***Escherichia/Shigella*** causally linked to menstrual disorders (OR = 1.004, P = 0.0003) — small effect per unit GRS but causally robust in a very large sample.
- ***Blautia*** causally protective (OR = 0.994, P = 0.038).
- Implicates the **estrobolome** and **β-glucuronidase–mediated estrogen recirculation** pathway — activating primitive 7.
- *Lachnospira*, *Anaerotruncus*, *Haemophilus* also show signals.

## Methods

Two-sample MR using MiBioGen microbiome GWAS and FinnGen menstrual-disorder GWAS. IVW with MR-Egger sensitivity testing.
