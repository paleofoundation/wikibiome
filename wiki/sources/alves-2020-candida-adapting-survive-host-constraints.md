# Key Findings

- **Lactate triggers beta-glucan masking while SCFAs (butyrate, acetate) cause unmasking** — the metabolic balance of the gut environment directly determines whether Candida is visible to the immune system ([[alves-2020-candida-adapting-survive-host-constraints]], expert-opinion).
- **Amino acid catabolism by Candida alkalinizes the phagosome**, inhibiting its maturation and enabling intracellular survival — a direct counter to the macrophage killing machinery.
- **Ubiquitination pathway rewiring** allows Candida to use multiple carbon sources simultaneously, unlike *S. cerevisiae* which uses them sequentially — a metabolic flexibility advantage in nutrient-variable host environments.

# Methods

Narrative review synthesizing experimental data on *C. albicans*, *C. glabrata*, and *C. auris* host adaptation strategies across multiple Candida species.
