# Hashimoto Thyroiditis and Clinical Outcomes of Papillary Thyroid Carcinoma

## Key Findings

- **PTC + HT patients** had significantly **lower risk** of lymph node metastasis (OR 0.787, p=0.001), central lymph node metastasis (OR 0.796, p=0.045), lateral lymph node metastasis (OR 0.845, p=0.02), distant metastasis (OR 0.435, p<0.001), extrathyroidal extension (OR 0.745, p<0.001), recurrence (OR 0.627, p<0.001), and vascular invasion (OR 0.718, p=0.004) vs. PTC-only.
- **Improved 20-year survival** (OR 1.396, p=0.005).
- **Higher risk** of multifocality (OR 1.245, p<0.001), perineural infiltration (OR 1.922, p=0.007), and bilaterality (OR 1.394, p=0.003) — consistent with diffuse lymphocytic thyroid infiltration pattern.
- 65 studies pooled across 32 case-control, 27 cohort, and 6 cross-sectional studies. Sensitivity analyses stable; Begg's test negative for publication bias.

## Methods

PubMed, Embase, Cochrane, Web of Science through December 2020. 1,992 records screened; 174 full-text; 65 included. Newcastle-Ottawa scale for cohort/case-control; JBI for cross-sectional. Random-effects model for I² ≥ 50%. Trial sequential analysis applied to mitigate false positives.
