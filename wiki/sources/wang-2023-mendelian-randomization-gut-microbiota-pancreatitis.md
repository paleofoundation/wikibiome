# Key Claims

- Multiple gut microbiota taxa are causally associated with four types of pancreatitis (acute, chronic, alcohol-induced AP, alcohol-induced CP) via bidirectional Mendelian randomization
- Class Melainabacteria (OR 1.801, p=0.008) has strong causality with alcohol-induced CP after Bonferroni correction
- Six genera causally linked to acute pancreatitis risk: Coprococcus3 and Eubacterium fissicatena group (increased risk), Prevotella9 and Ruminiclostridium6 (decreased risk)
- Reverse MR analysis showed pancreatitis itself causally influences gut microbiota composition, confirming bidirectional causation

# Methods/Findings

- **Study design**: Two-sample bidirectional MR using GWAS data from MiBioGen consortium (18,340 subjects, 211 taxa) and FinnGen (AP, CP, AAP, ACP outcomes)
- **Methods**: IVW as primary analysis; MR-PRESSO, Cochran's Q, MR-Egger for sensitivity; Bonferroni correction for multiple testing
- **AP results**: Coprococcus3 (OR 1.48), Eubacterium fissicatena group (OR 1.24) increased AP risk; Prevotella9 (OR 0.82), Ruminiclostridium6 (OR 0.70) decreased risk
- **CP results**: Defluviitaleaceae (OR 1.41), Barnesiella (OR 1.48), Eubacterium xylanophilum (OR 1.50), Sellimonas (OR 1.22) increased CP risk; Slackia (OR 0.63) decreased risk
- **Reverse MR (AP on microbiota)**: AP causally increased Proteobacteria, Lachnospiraceae NC2004 group, Marvinbryantia; decreased Holdemania, Oscillospira
- **AAP on microbiota**: Decreased Firmicutes, Tenericutes, Clostridia, Mollicutes, Coriobacteriaceae
