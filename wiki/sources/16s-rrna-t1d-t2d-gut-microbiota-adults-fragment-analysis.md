# Key Claims

- **Gut microbiota composition differs** between adult T1D, T2D, and healthy controls based on 16S rRNA fragment analysis
- **Distinct microbial signatures** identified for each diabetes type
- **Comparison of T1D and T2D microbiomes** reveals both shared and unique dysbiotic features
